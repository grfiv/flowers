NOVA Native Plants flowers table editor

This website will allow you to edit the flowers table directly.

Please note that once changes are made here the flowers.csv file will no longer be correct. The tables.html page can be used to download a flowers.csv file with the current information. This is a good idea for backup.

SEARCH: Notice that the search box searches all the fields not just the names.

ADD: Click the New button and fill in the fields. The new field will instantly appear in the app.

EDIT: Click on an entry and then click the Edit button. Changes will instantly appear in the app.

DELETE: ten cuidado!

New flower images and icons still have to go through the process of you inserting them into the screenshots and icons folders, respectively, and then me running a script to get them into shape and uploading them to the web.

Similarly, all the other *.csv files work as before with you editing the files and me loading them into the database.

This table displays only 10 items at a time. There is a page selector at the bottom of the page.

CAVEATS: (1) if any '_code' fields are empty or outside the range of numbers specified in the index table then the item will not appear in the app; tables.html will quickly show you any table. (2) you must not use double quotes ( " ), they will break the csv download; single quotes ( ' ) are OK. (3) "instant" updates may have to be preceded by a browser-cache flush.
